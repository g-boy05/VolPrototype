Ce projet utilise EntityFramework comme ORM, la base de données est créé avec l'approche codeFirst.
Avant de démarrer le projet :
 - Restaurer les packages nuget.
 - migration de la base deja creer sous le dossier migration.
 - Exécuter la commande update-database sous la console du gestionnaire de package. La commande permet de créer Snaposhot de la bdd sous votre serveur local sqlServer.

Le dossier testing contient 3 projets juste pour montrer les types de test qu'on peut mettre en place, il y'a juste un seul projet test unitaire pour la démo.
